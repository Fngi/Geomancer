### 鄉民風水師
這是一個利用資料視覺化的方式設計的人文地理工具，目前可查詢：
* 全台灣凶宅地點
* 台北市勞工局違反勞基法事業地點

成果可以在 [Google Play](https://play.google.com/store/apps/details?id=tacoball.com.geomancer) 下載
